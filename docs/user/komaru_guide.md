# KomaruPhone User Guide

## Включение/выключение
- Power button (GPIO17) - включение/выключение
- Долгое нажатие Power - принудительное выключение

## Навигация
- Volume Up/Down - навигация по меню
- Program button - дополнительная функция
- Навигационная панель на экране - Back/Home/Recents

## Основные приложения
- **KomaruBrowser** - веб-браузер
- **KomaruNotes** - заметки (Obsidian-style)
- **KomaruGallery** - просмотр фото
- **KomaruAppStore** - установка приложений
- **KomaruTerminal** - командная строка
- **KomaruSettings** - настройки системы

## Звонки и SMS
- Наберите номер в приложении Телефон
- Нажмите Power для вызова
- При входящем звонке - Power ответить, Program отклонить

## Подключение к Wi-Fi
1. Откройте Настройки -> Сеть
2. Включите Wi-Fi
3. Выберите сеть из списка
4. Введите пароль
